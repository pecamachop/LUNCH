Push: Subir el último commit de archivos realizado en nuestro ordenador al repositorio virtual.
Pull: Descargar la versión más reciente de archivos del repositorio virtual.

Cada Carpeta en el repositorio tiene un archivo de notas, que servirá para que el responsable de esa parte haga notas a sus compañeros, ponga sus dudas, o realice sus propios apuntes... (el archivo se encuentra vacío inicialmente).

Para empezar a trabajar con Git en ubuntu:

* Descarguen git.
* Creen una carpeta en la cual van a poner la información del repositorio.
* Marquen esa carpeta como repositorio.
* Enlacen esa carpeta con el repositorio remoto bajo el enlace: https://github.com/Joakoxdhalo/LUNCH.git
* Una vez enlazada la carpeta local al repositorio virtual, hagan Pull, se descargarán los archivos del repositorio.
* Una vez con todos los archivos en su repositorio local, suban sus archivos a la carpeta en la cual están trabajando.
* Hagan Push, para actualizar el repositorio local con los arhivos que acaban de poner en su repositorio local.


Recuerden:
* Antes de hacer Push, siempre deben hacer Pull, para actualizar el repositorio local con la última versión del repositorio virtual, para no perder archivos, luego si pueden hacer Push.
